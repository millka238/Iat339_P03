Project 1 — MaPastry Website

Year: 2026  |  Timeframe: 3 weeks  |  Tools: Figma, Visual Studio Code  |  Category: Web Design & Development

Design Intent

Create a simple, intuitive experience that allows users to easily explore the brand and take action. Focus on clear navigation, strong visual hierarchy, and a warm, cohesive aesthetic that reflects Maple Syrup Pastry.

Problem

Local bakery websites often lack clear structure and personality. Menus can be hard to browse, product details are limited, and the overall experience feels generic.

Solution

Strong Visual Identity — A warm, distinctive design system reflecting the maple syrup concept.

Simplified Navigation — Minimal, intuitive layout between menu, story, and ordering.

Focused Conversion Points — Clear calls to action like "Order now".

Brand Storytelling — Dedicated sections to communicate uniqueness and build trust.

Challenges

  • Creating a distinctive identity for a small, local bakery
  • Balancing aesthetic design with usability and clarity
  • Simplifying navigation while still presenting enough information
  • Designing for both storytelling and quick user actions

Outcomes

  • Developed a cohesive and recognizable visual identity
  • Improved user experience with clear structure and navigation
  • Created strong call-to-action points to support conversions
  • Delivered a modern, engaging website that highlights the brand’s uniqueness

Personal Reflections

This project helped me better understand the balance between design and usability. I learned how important clear hierarchy and navigation are in web design, and how thoughtful visual choices can strengthen a brand’s identity. It also improved my skills in Figma and front-end development, especially in turning a concept into a functional interface.





Project 2 — Landing Pages Design

A one-page landing design created as part of a 4-day intensive course. The goal was to design visually appealing and user-friendly landing pages for different companies.

Problem / Goal

Design a simple yet engaging landing page that attracts users and encourages sign-up — clean, consistent, and modern while reflecting a wellness theme.

Process

Step-by-step tutorials to practice layout, colour, typography, and understanding how small design decisions affect the overall user experience.

Challenges

  • Keeping the design simple without making it feel empty
  • Maintaining visual consistency across all sections
  • Balancing aesthetics with usability and clarity

Outcomes

The final landing page communicates simplicity and clarity, with a visually appealing layout and clear calls to action that guide the user through key sections.

Personal Reflections

This project helped me understand how color, typography, and spacing influence readability and user perception. I also improved my ability to iterate quickly and apply feedback effectively.
